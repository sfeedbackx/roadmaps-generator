# Folder structure
- verbs/*.py : for action implementations (scrap...)
- utils/ : for reusable modules like config and format 
- data/ : for static data however it could be deleted 
- static/ : frontend 
- main.py : for backend and for sure it will become a Folder
- agent_dev.ipnb(temporary) : to develop the agent 
# naming convention 
snake_case : var , function
UPPER_SNAKE_CASE : classes , const
